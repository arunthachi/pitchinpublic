# Pitch in Public Agent Pack

This folder is the persistent handoff pack for future coding agents working on Pitch in Public.

Read these files before making non-trivial product or architecture changes:

- [CONTEXT.md](CONTEXT.md) - current project context, environments, stack, routes, and operational notes.
- [MEMORY.md](MEMORY.md) - product decisions already made, rejected directions, and high-signal founder preferences.
- [PRODUCT_SPEC.md](PRODUCT_SPEC.md) - detailed product spec for founders, organizers, platform admin, and the pilot workflow.
- [WORKFLOWS.md](WORKFLOWS.md) - implementation workflows, QA expectations, release process, and route ownership.
- [ROADMAP.md](ROADMAP.md) - phased roadmap and what should remain manual until validated.
- [skills/pitch-in-public/SKILL.md](skills/pitch-in-public/SKILL.md) - project-local skill entry point for agents.

These docs are intentionally agent-facing, not marketing copy. Keep them current when shipping meaningful product changes.
